## Tech Stack

## Run Locally

Clone the project

```bash
  git clone [https://link-to-project](https://github.com/victoryanonymous/Layer0x.git)
```

for Backend :
```bash
   npm install
```

Go to the project directory frontend folder

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

To Start the server

from frontend

```bash
  npm start
```

from backend 
```bash
   node server.js
```
